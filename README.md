# UA SIM — Union Arena Simulator

Fan-made web simulator for the UNION ARENA trading card game.
เกมจำลองการ์ด UNION ARENA เล่นผ่านเบราว์เซอร์ (แนวนอน รองรับ PC และมือถือ)

**▶ เล่นเลย: https://bright4479.github.io/ua-sim/**

## Features
- 🃏 **Deck Builder** — การ์ดทั้งหมด 10,473 ใบ จาก 59 series, filter ตาม series / color / type / rarity / trigger, ตรวจกติกาจัดเด็คอัตโนมัติ (50 ใบ, 1 series, 1 สี, สูงสุด 4 ใบ/หมายเลข), save เด็คในเครื่อง
- ⚔️ **Single Player** — เล่น 1v1 กับบอท ตาม Official Rule Manual v1.1 ครบทุก phase (Start / Movement / Main / Attack / End), Raid, trigger check, keyword ครบ
- 🖱 ลากการ์ดจากมือลงสนาม, คลิก Deck เพื่อ extra draw, เปิดดูกอง Sideline/Removal ได้
- 🤝 **Co-Op (เชิญเพื่อน)** — กำลังพัฒนา

## Run locally
เปิด `index.html` ในเบราว์เซอร์ได้เลย หรือ `python -m http.server` แล้วเข้า localhost
